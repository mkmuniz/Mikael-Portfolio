# 💻 Front-End

## Setup

- Install `yarn` or `npm` package manager.
- Open a folder into `front` folder
- Type one of these commands below
```bash
# npm
npm install

# yarn
yarn install
```

## 🌐 Run Server
You can run the project with one of these commands below:
```bash
# npm
npm run dev

# yarn
yarn dev
```
